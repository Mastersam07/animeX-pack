**animeX** is a lightweight, Pythonic, dependency-free, library (and command-line utility) for downloading YouTube Videos.

**Behold, a blend of simplicity versus flexibility**::

    $ py animeX -m -v
    $ py animeX -m --version
    $ py animeX -m --name NAME

Features
--------
- Ability to Capture Thumbnail URL.
- Extensively Documented Source Code
- No Third-Party Dependencies
- Saves video to local device
